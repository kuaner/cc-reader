CC Reader

A native macOS viewer for Claude Code and Codex session history. Browse timelines, inspect tool usage, and resume sessions — all locally.

Download Source on GitHub
v1.6.4 macOS 14+ MIT Universal
Features
Built for daily use
01

Multi-pane layout

Split horizontally or vertically, up to 12 panes per tab. Compare sessions side by side like a tiling terminal.

02

Claude + Codex

Unified viewer for both Claude Code and OpenAI Codex. Toggle sources in the sidebar with a single click.

03

Rich timeline

Markdown rendering, syntax highlighting, thinking blocks, tool usage, and structured diffs in a native timeline.

04

Real-time sync

FSEvents monitoring with debounced incremental parsing. New messages appear as sessions evolve.

05

Session picker

Quick-search overlay with keyboard navigation. Jump to any session across all your projects.

06

11 color themes

Everforest, Catppuccin, Dracula, Gruvbox, Nord, and more. Editor-inspired, easy on the eyes.

07

Fast indexing

Lightweight first-launch scan with lazy metadata warmup. Opens large histories without blocking.

08

Resume commands

Copy claude --resume or codex resume directly from the toolbar. Pick up where you left off.

09

Localized

English, Japanese, and Simplified Chinese. Confirmation detection across all three languages.
